## 🛠 实现的功能

### 1. NotificationManager 单例
- **文件**: `~/Desktop/ios-VitaMind/Sources/Services/NotificationManager.swift` (142 行)
- **设计**: `@MainActor` 隔离, `ObservableObject`, 单例
- **3 类每日通知**:
  - 09:00 `daily_pull` — "🎴 Today's card is ready"
  - 20:00 `daily_habit` — "⏰ Habit check-in"
  - 22:00 `streak_rescue` — "🔥 Last call" (仅当 habitCount > 0)
- **Trigger**: `UNCalendarNotificationTrigger` 重复 daily
- **持久化**: `@AppStorage("vita_notifications_enabled")` 用户开关

### 2. 接入 GameState
- `bootstrapNotifications()` — 启动时申请权限 + 排程
- `applyNotificationPreferenceChange()` — 习惯数变化时重排
- `pullDailyCard()` 末尾 rescheduleAll(habitCount:)

### 3. App 启动集成
- `VitaPocketApp.swift` 用 `.task` 调 `gameState.bootstrapNotifications()`
- 不用 `init()` 避免 @StateObject 初始化竞态

### 4. Settings Tab UI
- Notifications section: 开关 + 副标题 + "Send Test Notification" 按钮
- 状态文案: 0/1/2/3 reminders based on auth status
- Footer 解释: "Three gentle nudges: 9 AM (daily card), 8 PM (habit check-in), 10 PM (streak rescue). Local only — no server."

### 5. **关键 Bug 修复**: UNUserNotificationCenterDelegate
- 模拟器 simctl push 通知**前台不显示 banner** (iOS 标准行为)
- 必须实现 `willPresent(_:withCompletionHandler:)` 返回 `[.banner, .sound, .badge]`
- 修了"开发时用 simctl push 测得通、真机却看不到 banner"的经典坑
- **副作用好处**: 真实用户点 "Send Test Notification" 按钮也能看到 banner, 不会以为功能坏了

### 6. 测试按钮 (`sendTestNotification`)
- `UNTimeIntervalNotificationTrigger(timeInterval: 1, repeats: false)`
- 1 秒后弹出, banner + sound

## 🎬 GIF 制作工具链 (沉淀)

```bash
# 录屏
xcrun simctl io <device> recordVideo --codec h264 /tmp/demo.mp4 &

# 测试 (XCUITest 触发完整流程)
xcodebuild test -project VitaPocket.xcodeproj -scheme VitaPocket \
  -only-testing:VitaPocketUITests/NotifDemoTests
kill -INT %1   # 停录屏

# 转 GIF (palette 优化色彩)
ffmpeg -ss 12 -t 18 -i /tmp/demo.mp4 \
  -vf "fps=10,scale=420:-1,palettegen" /tmp/pal.png
ffmpeg -ss 12 -t 18 -i /tmp/demo.mp4 -i /tmp/pal.png \
  -filter_complex "fps=10,scale=420:-1,paletteuse" out.gif
```

## 🐛 沿途踩的坑 (SOUL.md 自我进化素材)

1. **cliclick 没法点 Simulator** → 改用 XCUITest 内部 tap
2. **Foreground 不显示 banner** → 加 `UNUserNotificationCenterDelegate.willPresent`

5. **xcodebuild 启动慢** → trim 跳过前 12s
6. **fGif 太大 (4MB)** → 减小 fps 10 + 缩 width 420 → 1.5MB
