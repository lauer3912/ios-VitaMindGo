## 1. 商业逻辑 (Why)

### 1.1 财务背景
- 佛罗多老爷有 **$50,000 美元贷款**, **~4 个月**内要还
- VitaMindGo v3.0.0 上架 = 0 收入 (Free tier)
- 需要 **可持续现金流**, 而非一次性买断

### 1.2 为什么是订阅, 不是一次性 IAP

| 维度 | 订阅制 ✅ | 一次性 IAP ❌ |
|------|----------|---------------|
| LTV | 持续 ($4.99 × N 月) | 一次 ($9.99) |
| 现金流可预测性 | 高 (每月自动) | 低 (看新用户) |
| 与 App Store 心理价位 | $4.99/月是健康类主流 | $9.99 在 habits 类偏高 |
| 老爷 4 个月还贷 | ✅ 持续可预测 | ❌ 卖完就没 |
| 文档建议 ("订阅制 + 可量化价值") | ✅ 强匹配 | 弱匹配 |

### 1.3 VitaMind 在 2026-2028 趋势里的卡位
- **方向 3**: 循证心理健康与数字福祉 (Apple Health 集成 + AI Coach + 引用源)
- **方向 7**: 预防性健康与慢病自我管理 (HealthKit + Apple Watch + 心脏/睡眠)
- 文档强调"端侧优先"+"透明度即信任" → VitaMind 引用源系统 6 层防护已就位

## 2. 产品形态 (What)

### 2.1 定价矩阵 (Freemium 订阅)

| 档位 | 价格 | 内容 | App Store Connect ID |
|------|------|------|---------------------|
| **Free** | ¥0 | 习惯追踪 + 5 条 AI 对话/天 + 基础卡片 | — (默认) |
| **Pro Monthly** | **$4.99/月** | 无限 AI 对话 + 高级洞察 + Watch 复杂功能 + 限定卡牌 | `vitamind_pro_monthly` |
| **Pro Yearly** | **$39.99/年** (省 33%) | 同上 | `vitamind_pro_yearly` |
| **Pro Trial** | 免费 7 天 | Pro 全部功能, 到期自动转 Monthly | (StoreKit 2 Introductory Offer) |

**为什么 $4.99 不是 $2.99 / $9.99**:
- 行业基准: Streaks $4.99, Way of Life $4.99, Headspace $12.99 (高端)
- VitaMind 卡中端 ($4.99 是 habit+AI 健康类的 sweet spot)
- 年付 $39.99 实际 $3.33/月 (跟 Monthly 比省 33%, 引导年付转化)

### 2.2 Free vs Pro 边界 (硬约束, 不能模糊)

| 功能 | Free | Pro |
|------|------|-----|
| 习惯追踪 (Habit CRUD + Streak) | ✅ 无限 | ✅ 无限 |
| HealthKit 数据读取 (心率/睡眠/活动) | ✅ | ✅ |
| 基础卡片 (Common + Rare) | ✅ | ✅ |
| AI Coach 对话 | ⚠️ 5 条/天 | ✅ 无限 |
| 高级洞察 (心率变异性趋势 / 习惯坚持预测) | ❌ | ✅ |
| Apple Watch 复杂功能 (Complication + Smart Stack) | ⚠️ 基础 | ✅ 高级 |
| 限定卡牌 (Legendary + 限定主题) | ❌ | ✅ |
| 历史数据导出 (CSV/PDF) | ❌ | ✅ |
| 多个 Theme 包 | ❌ | ✅ |

### 2.3 关键 UX 原则
- **不藏付费墙**: 5 条/天限制在第 5 条用完后立即显示 paywall, 不藏
- **7 天试用**: paywall 默认勾选 Yearly + "Start Free Trial" 按钮
- **价格透明**: 显示"$4.99/月, $39.99/年 (省 33%)", 不用小字
- **可一键取消**: Settings → "Manage Subscription" 直接跳转 App Store

## 3. 技术架构 (How)

### 3.1 StoreKit 2 (iOS 15+)

```swift
// 核心 API
import StoreKit

// 1. 加载产品
let products = try await Product.products(for: ["vitamind_pro_monthly", "vitamind_pro_yearly"])

// 2. 购买
let result = try await products[0].purchase()

// 3. 验证 (Transaction 自动验签)
switch result {
case .success(let verification):
    switch verification {
    case .verified(let transaction):
        // 启用 Pro
        await transaction.finish()
    case .unverified:
        // 验签失败, 不启用
        break
    }
case .userCancelled:
    break
}

// 4. 监听交易更新 (跨设备, App 重启)
for await update in Transaction.updates {
    // 处理
}
```

**为什么选 StoreKit 2, 不是 StoreKit 1**:
- 现代 Swift Concurrency API
- 自动 JWS 验签 (减少后端依赖)
- Transaction 跨设备同步 (用户换设备自动恢复)
- 官方推荐 (iOS 15+), Apple 审核也对现代 API 更友好

### 3.2 客户端 vs 服务端验证

| 方案 | 优点 | 缺点 | 决策 |
|------|------|------|------|
| 纯客户端 (StoreKit 2 JWS) | 零后端, 快速上线 | 无法防止越狱, 收入数据需另查 | ✅ **先用这个** |
| App Store Server Notifications V2 | 实时订阅状态 webhook | 需要公网 URL, 没后端 | ⚠️ 暂不启用, 后续加 |
| 自建后端 + App Store Server API | 完整控制 | 开发成本高 | ❌ 暂不需要 |

**Phase 1 决策**: 纯客户端 StoreKit 2 + iTunes Lookup 公开数据
**Phase 2 (可选)**: 加 App Store Server Notifications V2 (设 URL 但不接, 留扩展)

## 5. 风险与对策

### 5.1 App 审核新雷区 (Health + Subscription)

| 雷区 | Guideline | 对策 |
|------|-----------|------|
| 付费墙黑箱 | **3.1.2** | 明确显示价格 + 续费条款 + 取消方式 |
| 自动续费没披露 | **3.1.2(a)** | paywall 加 Apple 标准 EULA 链接 + 续费条款 |
| 退款体验差 | **2.1** | Settings 加 "Manage Subscription" (跳 Apple ID 设置) |
| 医疗声明 + 订阅 | **1.4.1** (续) | 延续"非医疗建议"措辞, Pro 不解锁新医疗功能 |
| 价值不明显 | **4.0** | 截图 + 描述必须明确 Pro 解锁什么 |

### 5.2 EU AI Act 合规
- VitaMind 走 **lifestyle/wellness** (低风险), 不重新分级
- "Pro 解锁" 不涉及 AI Act 风险升级 (因为 AI Coach 已经在 Free)
- 隐私政策保留 "On-device AI, no account, no tracking" 措辞

### 5.3 收入影响评估 (保守估计)

| 指标 | 假设 | 月收入 |
|------|------|--------|
| 上架首月下载 | 100-500 | — |
| Free → Pro 转化率 | 2-5% (健康类行业基准) | — |
| 月活 Pro 用户 (3 个月后) | 20-50 | $100-$250 |
| 6 个月后 (持续增长) | 50-200 | $250-$1000 |

> ⚠️ 这是保守估计, 实际取决于 marketing + 留存 + 评价
> 4 个月还贷 $50,000 ÷ 4 = $12,500/月需要, **订阅不够, 需配合其他项目** (液态玻璃 + App Intents + 第 2 个 App)

## 8. 关键决策记录 (为什么这样选)

| 决策 | 选 | 不选 | 原因 |
|------|----|----|------|
| API | StoreKit 2 | StoreKit 1 | 现代, 自动验签, iOS 15+ 是 2020 发布的, 覆盖率 >99% |
| 价格 | $4.99/月 | $2.99 / $9.99 | 行业 sweet spot |
| 模型 | Freemium 订阅 | 一次性 IAP / 广告 | 还贷需持续现金流 |
| 试用 | 7 天 | 14 天 / 30 天 | 7 天是行业标准, 不会吓跑用户 |
| Family Sharing | 暂不开 | 开 | 苹果抽成 + UI 复杂, 先看数据 |
| 后端 | 纯客户端 | 自建后端 | 节省 4 周开发时间, Phase 2 再加 |
| 服务端通知 | 暂不接 | 接 | 没后端接不了, 后续 |
| Paywall 默认选项 | Yearly | Monthly | 年付 LTV 高 12 倍, 默认勾引导转化 |
