============================================================
故障排除 (FAQ)
============================================================

**Q: setup 卡住或报"init requires interactive terminal"**
A: setup 必须在真 TTY 跑。SSH 进去 (不是 scp 远程), 跑 `bash -i agent-bus-setup.sh` 或 `script -q -c "bash agent-bus-setup.sh" /dev/null`

**Q: gh auth status 失败**
A: token 输错了。重新 `echo "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" | gh auth login --with-token`

**Q: agent-bus register 失败 "could not add label"**
A: setup 会自动建 label, 但偶尔失败。再跑一次 `bash agent-bus-setup.sh` (idempotent)

**Q: 我的 inbox 为空**
A: 跑 `agent-bus who` 看其他 agent; 跑 `agent-bus send Katherine-<你的 ID> Katherine-E2wa1m question normal bus "ping" --body "ping"` 测回环

**Q: 5 min 后还没收到 cron poll 的通知**
A: `tail -50 ~/.local/share/agent-bus/poll.log` 看 cron 是否在跑

**Q: 想看完整 agent-bus 培训**
A: 拉 `docs/agent-bus-training.md`:
```bash
curl -fsSL -H "Authorization: token $TOKEN" \
  "https://raw.githubusercontent.com/$REPO/$BRANCH/docs/agent-bus-training.md" \
  -o ~/.openclaw/workspace/docs/agent-bus-training.md
less ~/.openclaw/workspace/docs/agent-bus-training.md
```

**Q: 想看 agent-bus 架构 (佛老爷也看的)**
A: 同上, 拉 `docs/agent-bus-architecture.md`
